# 🏠 집 가구싶어 (SweetHome)

React, TypeScript, Rest API를 활용한 라이프 스타일 홈 데코·인테리어 쇼핑몰 프로젝트입니다.
<br />
성별과 연령에 상관없이 모두가 관심을 가지고 이용할 수 있는 인테리어 관련 주제로 선정하였으며,
<br />
‘집’ + ‘가구’ + ‘가고싶다.’ 의 합성어로 언제 어디서나 집에 가고싶게 만드는 제품을 판매하다는 의미의 쇼핑몰입니다.

<br />

### 프로젝트 상세 기능

## Header
- 카테고리 (Navbar)
- 제품 검색 기능

### 메인 페이지 (Home)
- 이미지 슬라이더

## 상품 페이지 (Shop)
- 상품 목록 페이지
- 상품 상세 페이지
- 장바구니 담기 기능
- 결제 페이지

## 장바구니 페이지 (Cart)
- 장바구니 목록 조회
- 선택 상품 구매
- 전체 상품 구매

## 결제 페이지 (Buy)
- 결제 기능

## 마이페이지 (Mypage)
- 주문 내역 조회
  - 상품 구매 취소
  - 상품 구매 확정
  - 상세 정보 조회
- 계좌 관리
  - 전체 계좌 조회
  - 계좌 등록
  - 계좌 삭제
- 개인 정보 수정

## 관리자 페이지 (Admin)
- 전체 사용자 목록 조회
- 상품 관리
  - 전체 상품 목록 조회
  - 상품 등록
  - 상품 삭제
  - 상품 정보 수정
- 주문 내역 관리
  - 전체 거래 내역 조회
  - 거래 취소
  - 거래 확정
  
## 인증 & 인가
- 회원가입
- 로그인
- 로그인 인증
- 로그아웃

## 프로젝트 테스트

### clone project

```bash
$ git clone git@github.com:FE5-TEAM9/sweethome.git
```

### go to project

```bash
$ cd sweethome
```

### install npm

```bash
$ npm install
```

### start project

```bash
$ npm run dev
```
